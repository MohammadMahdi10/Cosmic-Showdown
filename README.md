# Cosmic Showdown - 2D Fighting Adventure Game

## Summary

A 2D action fighting game where the player selects levels, battles enemies, collects coins, and uses potions to gain powerups. The player must defeat enemies, complete objectives, collect resources, and unlock the final boss level. Potions can help the player by restoring health or increasing speed, making it easier to survive difficult battles. The goal is to progress through the levels, grow stronger through experience and coins, and defeat the boss in the final challenge.

## Features

- Classic 2D fighting and platform gameplay.

- Fight enemies across different levels.

- Collect coins to increase your total coin count.

- Collect potions for health and speed boosts.

- Use the in-game shop to buy useful potions.

- Gain experience by attacking and defeating enemies.

- Level up to increase player strength.

- Unlock the boss level by completing earlier levels.

- Includes a main menu, level selector, settings screen, pause menu, shop, win screen, lose screen, and draw screen.

## How to install the game

1. Make sure you have Python installed. Python 3.13 is recommended.

2. Download the project ZIP file and extract it to a folder on your computer.

3. Open PowerShell.

4. Go into the extracted project folder. For example:

```powershell
cd "C:\Users\shand\Downloads\NEA-20260616T130630Z-3-001\NEA"
```

5. Install pygame:

```powershell
py -3.13 -m pip install pygame
```

5. If you are using Python 3.14 and pygame does not install, use pygame-ce instead:

```powershell
py -m pip install pygame-ce
```

6. Run the game:

```powershell
py main.py
```
